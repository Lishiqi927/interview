String不属于基础的数据类型。
基础类型一共有八种，这八种又分为四种类型，分别是整型、浮点型、字符型、布尔型，整型有byte、short、int、long，浮点型有double、float，字符型有char，布尔型有boolean。byte占8字节，short占16字节，int占32字节，long占64字节，double占64字节，float占32字节。

char
